Accessible

To test out keyboard accessibility:

  1. Set focus on this element by clicking on it or keyboard tabbing to it.
  2. Press the right arrow key to navigate to the next panel.
  3. Or press the left arrow key to navigate to the previous panel.

Now, let's check how things look with JavaScript disabled:

  1. Open your browser's preferences.
  2. Disable JavaScript.
  3. Refresh the page.
  4. All content should be right there.
